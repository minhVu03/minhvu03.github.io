What I've coded


Carrota

 

A Discord bot programmed in Python using the discord.py API. Carrota responds to some random text messages in a server such as "ping" or any messages that contain "food". It meows back at you if you meow. It also responds to various commands that start with '$'. Carrota was born in 2021 and is still being updated as more features are added. My goal is to create a personality for it! Carrota falls asleep if it's inactive for a long time :( so UptimeRobot pings it every once in a while to keep it awake!



Wanderlust

 

A travel recommendation application programmed in Java. I created this project because one of my goals is to be able to see all the pretty places in the world, and to experience new cultures by meeting new people. I think that the hardest part about travelling is coming up with a plan and deciding on which places to visit. Wanderlust lets you write reviews about cities and saves them to a database of other people's reviews from previous sessions. You can then search for all the reviews of a particular city. Check out the README on my Github to see what else you can do with it!



A Walk in the Park

 

A 2D platform game programmed in C using the Allegro graphics library. The game is set in a post-apocalyptic city where the environment has been destroyed. You are now a bunny trying to survive in a park by dodging trash, toxic fumes, and chopped-down trees! Make sure to have your audio on when playing!
